Fundamentals

What is a Stack?

Understanding the core concept behind LIFO data management.

Definition

A Stack is a linear data structure that stores items in a Last-In, First-Out (LIFO) manner. Elements are added and removed from the same end, called the TOP.

Memory Layout

Elements occupy contiguous memory (array-based) or linked nodes. A TOP pointer always tracks the most recently added element.

Single Access Point

Unlike arrays, you can only access the TOP element. To reach any other element, you must remove all elements above it first.

Context & Usage

Real-World Analogies

Everyday examples to understand the Last-In, First-Out (LIFO) principle.

Stack of Plates

In a cafeteria, clean plates are stacked vertically. You always take the top plate first. The last plate placed on the stack is the first one removed.

LIFO: Physical

Stack of Books

Books piled in a box or on a table. To access the bottom book, you must remove all books above it one by one. Direct access is blocked.

LIFO: Physical

Browser History

The 'Back' button pops the last visited URL from the history stack. The current page is always at the TOP of the stack.

LIFO: Digital

Undo / Redo

Text editors store your actions in a stack. Pressing Ctrl+Z pops the most recent action to reverse it.

LIFO: Digital

Practical Uses

Applications of Stack

Common algorithmic problems solved efficiently using LIFO.

String Reversal

Push characters onto the stack one by one, then pop them to reconstruct the string in reverse order.

Parenthesis Matching

Validate balanced brackets in code (e.g., matching { with }) by pushing openers and popping on closers.

Recursion Handling

Compilers use the system "Call Stack" to keep track of function calls, local variables, and return addresses.

Postfix Evaluation

Evaluate mathematical expressions (like 3 4 +) directly by processing operands and operators linearly.

Infix to Postfix

Convert standard expressions A+B into Reverse Polish Notation AB+ for easier machine evaluation.

Infix to Prefix

Convert standard expressions A+B into Polish Notation +AB, often used in parsing algorithms.

Chapter Summary

Key Takeaways: STACK

LIFO Principle

Stack is a linear data structure following Last-In-First-Out. Elements are inserted and deleted from only one end called TOP.

Core Operations

PUSH: Insert element (check Overflow).
POP: Remove element (check Underflow).
PEEK: Access top element without removal.

Implementation

Can be implemented using Arrays (static size, contiguous memory) or Linked Lists (dynamic size, pointer-based).

Key Applications

Used in Recursion, String Reversal, Parenthesis Matching, and Expression Conversion (Infix to Postfix/Prefix).

Towers of Hanoi

A classic recursive problem solved using a stack concept. Minimum moves required for n disks is 2n - 1.
